Why should you ensure that the script element that links to the jQuery library file comes before any other script element for a .js file that includes jQuery code?

Why should you avoid including jQuery in a project if you don't need it?
When targeting only modern browsers, it is better to avoid using jQuery's backward-compatible features.

How does every jQuery statement start?
Every jquery statement begins with a $. The $ symbol specifies that any code that follows should be interpreted using the jquery library

What is a Content Delivery Network (CDN)?  List and explain at least FOUR advantages in using a CDN.
CDN is a large distributed system of servers deployed in multiple data centers across the internet. Some advantages of using CDN are that it can be distributed to data centers, has built in version control, has usage analytics and a high capacity infrastructure.
